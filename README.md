# Dashboard de Vendas - Power BI

Projeto desenvolvido durante o desafio da DIO utilizando Power BI Desktop.

## Objetivo

Construir um dashboard de vendas utilizando a base Financial Sample.

## Dashboard

O projeto apresenta:

- Sales por País
- Units Sold por País
- Profit por País
- Profit por Segmento

## Tecnologias

- Power BI Desktop
- Microsoft Bing Maps
- DAX
- Data Visualization

## Dataset

Financial Sample
